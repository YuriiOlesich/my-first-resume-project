# Resume Website

A personal resume website developed with HTML and CSS to showcase my frontend development skills, professional experience, technical expertise, and completed projects.

## 🚀 Overview

This project demonstrates my ability to build clean, responsive, and accessible web interfaces using modern HTML and CSS practices. The website is designed to provide recruiters and potential employers with a clear overview of my qualifications and portfolio.

## 🛠 Tech Stack

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Web Design

## ✨ Features

- Responsive layout for desktop, tablet, and mobile devices
- Structured and semantic HTML markup
- Clean and maintainable CSS architecture
- Optimized images and performance
- Professional resume presentation
- Project portfolio section with live demos and source code links
